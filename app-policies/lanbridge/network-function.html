Network Share Network Function Statement

Network Share is not a VPN and does not bypass system or network restrictions.

Key points

  • Not a VPN: LAN proxy/forwarder within the same Wi‑Fi/hotspot; no encrypted tunnel service.
  • No bypass: Does not circumvent firewalls, carrier limits, or content blocks; clients must set proxies themselves.
  • Traffic path: Stays inside LAN; if host runs its own VPN, traffic may traverse that VPN under host control.
  • No cloud relay: We do not operate external relay servers.
  • User responsibility: Use lawfully; do not misuse to bypass restrictions.
  • Contact: wenyuanshuai@gmail.com

要点

  • 非 VPN：仅同一 Wi‑Fi/热点内做局域网代理/转发，不提供加密隧道。
  • 不绕过：不绕过防火墙/运营商/内容封锁，客户端需自行配置代理。
  • 流量路径：流量留在局域网；若主机开启自有 VPN，流量可在其控制下经由该 VPN。
  • 无云中继：不提供外部中继服务器。
  • 用户责任：合法合规使用，禁止用于绕过限制。
  • 联系：wenyuanshuai@gmail.com
